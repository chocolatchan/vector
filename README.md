# Vector 
- Implementation of vector in C programming language.
- Using doubly linked-list structure as core.


---
### For running test:
  - **Linux | Unix-like systems**
  ```bash
  gcc test.c vector.c -C -o test
  ./test
  ```
  - **Windows**
  ```cmd
  gcc test.c vector.c -C -o test.exe
  ./test.exe
  ```
